Data Files for Exp/Log Interp:

scenes/tableVase.sc : This is the scene file. It references several meshes from the meshes/ folder.

interp/Rotate.txt : This file contains the keyframe information used in the rotate example (figure 5 in the paper)

interp/Cop.txt : This file contains the keyframe information used in the center of projection example (figure 6 in the paper)

interp/interp3.txt : This file contains the keyframe information for an example which shows that our large rotation fix works. (The rotation between keyframes 2 and 3 is < pi/2.)
